Archlinux系统的安装

准备环境

准备镜像：镜像下载地址：https://archlinux.org/download/

准备工具：实体机或虚拟机（VMware），U盘，UltraISO软件，下载好的镜像文件

还要看电脑是UEFI模式的还是BIOS模式的 这个可以在主板中找到

首先插上U盘（U盘文件一定备份好了这个过程U盘将被格式化）——>打开软件UltraISO–>继续使用–>左上角文件打开（找到你下载的Archlinux.iso文件）打开–>选择启动(B)–>写入硬盘镜像—>硬盘驱动器选择U盘–>点击写入等待完成即可

开始安装

不会怎么选择U盘启动的根据自己电脑品牌自行百度

进入到这个界面选择第一个回车（如果这个界面进不去可能是模式不对开机F2进入主板设置UEFI模式重新进入）

验证启动模式

这个目录如果存在大量文件说明你的系统是UEFI启动的，如果不存在说明系统可能是BIOS或CSM模式启动模式（清楚自己启动模式注意下面安装步骤不同模式的不通安装方法）

一般sda是我们的硬盘

如果磁盘存在分区需要把原有分区删除（提前备份磁盘数据）我的磁盘是sda：50G

我这里的分区方案

UEFI分区方式

分区名称 设备标号 磁盘格式
根 /dev/sda1 剩余空间默认格式
swap /dev/sda2 linux swap
home /dev/sda3 默认
boot /dev/sda2 EFI system

BIOS分区方式不需要boot分区

分区名称 设备标号 磁盘格式
根 /dev/sda1 剩余空间默认格式
swap /dev/sda2 linux swap
home /dev/sda3 默认

分区命令cfdisk

swap分区时内存的2倍左右 boot分区500M左右 home目录自行分配不要台小 剩余的给根目录

最下面的行时分区工具行

Delete 是删除分区键盘上下键可以调节你要删掉的分区

Resize 顾名思义就是重置，清除操作

Quit 是退出

Type 是选择分区的格式 linux swap 和 EFI system 就是在这个里面选择

Write 是保存写入

修改镜像源

vim /etc/pacman.d/mirrorlist

输入/China 找到中国的源回车 按n查找下一个找到tuna.tsinghua.edu.cn的源 按2dd 之后按gg到首行

找个空位置p粘贴上去 之后在下面 加上 按i 找到下一行输入

按ESC键 之后按shft + :（记得英文输入法）wq 保存退出
